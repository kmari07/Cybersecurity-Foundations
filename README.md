# Cybersecurity-Foundations
This repository contains a written overview of core cybersecurity concepts such as Cybersecurity threats, the CIA Triad (Confidentiality, Integrity, and Availability), and common cyber attacks.
## Purpose
This project was created to demonstrate foundational cybersecurity knowledge.
## Topics Covered
Some topics covered were Threat modeling, Confidentiality, Integrity, Availability, Phishing, Malware, Denial of Service Attacks.
